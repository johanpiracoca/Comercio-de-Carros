# Comercio de Automoviles
Espacio en el que se ubican los archivos para la creación de una Pagina Web referente a la comparación de las ventas de automoviles de diferentes marcas. En el aplicativo puedes descargar los datos utilizados para hacer los respectivos analisis y una muestra de los mismos. En complemento, puedes observar algunos conceptos de la Estadistica Descriptiva, aplicada a los datos estudiados como lo son: Graficos de barras para la distribucion de las ventas de carros por color, un histograma para analizar una variable especifica del dataset y un grafico de dispersion para analizar la correlacion entre dos variables. 

El presente aplicativo se desarrolla en el marco academico para conocer los fundamentos basicos del desarrollo web, por lo que se tuvo que conocer y aplicar los conocimientos de los siguientes temas:

1) Linea de comandos en terminales locales
2) Entornos de Desarrollo Globales e Individuales
3) Git y GitHub 
4) Python Intermedio
